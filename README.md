# Pharmacy Data Quality & Inventory Reporting

Business-oriented pharmacy inventory and data quality management project focused on validation workflows, inventory monitoring, reporting, and operational data consistency.

## Project Overview

This project demonstrates how structured inventory validation and reporting workflows can improve operational reliability and data consistency in pharmacy-related business environments.

The project focuses on:

* Inventory data validation
* Low stock monitoring
* Expiration date checks
* Duplicate detection
* Missing supplier information
* Inventory reporting
* Data quality KPIs

## Key Validation Checks

* Duplicate medicine records
* Missing supplier information
* Invalid expiration dates
* Inconsistent category naming
* Low stock monitoring
* Missing storage conditions
* Inactive or expired inventory detection
