# Pharmacy Data Quality & Inventory Reporting

Business-oriented data quality and inventory monitoring project for a small pharmacy environment.

## Project Overview

This project demonstrates how structured validation workflows, reporting processes, and inventory monitoring can improve operational reliability and data consistency in pharmacy-related business environments.

The project focuses on:

* Inventory data validation
* Duplicate detection
* Expiration date monitoring
* Missing supplier information
* Stock level reporting
* Data quality KPIs

## Technologies

* Python (Pandas)
* SQL
* Excel
* Power BI

## Key Validation Checks

* Duplicate medicine records
* Missing supplier information
* Invalid expiration dates
* Inconsistent category naming
* Low stock monitoring
* Inactive inventory detection

## Business Value

This project demonstrates how structured inventory data management and quality validation workflows can support more reliable operational processes and improve reporting quality.
